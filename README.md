# wm_colab

Projet de travail pour expérimenter un world model de type LeWorldModel sur un gridworld isométrique, avec développement local et exécution sur Google Colab.

## Objectif

Construire un repo propre avec :

- un environnement simple en rendu isométrique
- un pipeline de génération de dataset
- un notebook d'entraînement compatible Colab
- une organisation claire entre code source et notebooks

## Structure visée

```text
wm_colab/
├── README.md
├── requirements.txt
├── .gitignore
├── src/
│   ├── env.py
│   └── model.py
├── wm_dataset.ipynb
└── wm_train.ipynb
````

## Workflow

* développement en local
* versionning avec Git et GitHub
* exécution sur Google Colab
* stockage des artefacts dans Google Drive
